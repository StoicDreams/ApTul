## How it works

<webui-page-segment elevation="10">
- **Input SVG Path**: Paste your SVG path data into the input field.
- **Specify Rotation Angle**: Enter the angle in degrees by which you want to rotate the path.
- **Get Rotated Path**: The tool calculates the new coordinates and outputs the rotated SVG path data.
</webui-page-segment>
